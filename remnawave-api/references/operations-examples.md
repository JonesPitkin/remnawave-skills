# Operations Examples

Источник:

- `https://docs.rw/api/`

## Минимально подтвержденные operation examples

- `post /api/tokens`
- `get /api/tokens`
- `delete /api/tokens/{uuid}`
- `post /api/users`
- `patch /api/users`
- `get /api/hosts`
- `post /api/hosts`
- `patch /api/hosts`
- `get /api/config-profiles`
- `get /api/remnawave-settings`
- `post /api/system/tools/happ/encrypt`
- `post /api/system/testers/srr-matcher`
- `get /api/system/stats/recap`

## Как использовать этот файл

- Как карту существования operation groups.
- Не как frozen schema reference.
- Для точных request/response DTO, headers и field names открывать official interactive spec перед выполнением mutation.
