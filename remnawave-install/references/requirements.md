# Requirements

Источник:

- `https://docs.rw/docs/install/requirements/`

## Panel

- OS: рекомендованы `Ubuntu` или `Debian`.
- RAM: минимум `2 GB`, рекомендовано `4 GB`.
- CPU: минимум `2 cores`, рекомендовано `4 cores`.
- Storage: `20 GB`.

## Node

- OS: рекомендованы `Ubuntu` или `Debian`.
- RAM: минимум `1 GB`.
- CPU: минимум `1 core`.

## Software

- Panel и Node требуют `Docker` с `Docker Compose plugin`.
- Официальная requirements page показывает install command для Docker, но этот файл фиксирует только сам факт требования Docker, без вынесения внешней Docker-ссылки в local reference.
