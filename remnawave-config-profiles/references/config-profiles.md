# Config Profiles

Источник:

- `https://docs.rw/docs/learn-en/config-profiles/`

## Подтвержденные факты

- Config Profile — это full `Xray-core` configuration template for a Node.
- В editor можно:
  - добавлять `Inbounds`
  - удалять `Inbounds`
  - изменять другие Xray settings
- Официальный guide демонстрирует `Load from GitHub`.
- В profile list:
  - левая иконка показывает число `Inbounds`
  - правая иконка показывает число `Nodes`, использующих профиль

## Ограничение guide

- Официальный текст прямо говорит, что статья не покрывает конфигурирование `Xray-core` itself.
