# Happ Routing

Источник:

- `https://docs.rw/docs/learn-en/quick-start/`

## Что подтверждено

В navigation menu docs описывают раздел `Utilities`, где находится:

- `Happ Routing` — rule builder для создания и управления routing rules для Happ.

## Что не подтверждено в разрешенном наборе источников

В перечисленных пользователем официальных источниках нет отдельной полной инструкции со всеми полями, экспортными форматами и примерами правил Happ Routing.

Следовательно:

- можно подтверждать existence и purpose;
- нельзя придумывать недокументированную пошаговую механику.
