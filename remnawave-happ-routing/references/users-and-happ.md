# Users And Happ

Источник:

- `https://docs.rw/docs/learn-en/users/`

## Связь Users и Happ

Официальная user management page говорит, что в `Detailed Info` присутствуют:

- `VLESS UUIDs`
- `Happ Crypto links`

Это подтверждает, что Happ-related user data действительно является частью Remnawave user workflows.
