# Initial Setup

Источник:

- `https://docs.rw/docs/learn-en/quick-start/`

## Что относится к Users

- После установки Panel первый зарегистрированный пользователь становится `super-admin`.
- Если пароль забыт, docs указывают использовать Rescue CLI:
  - `docker exec -it remnawave remnawave`
- В navigation menu users management находится в `Management`.

## Навигация

- `Management` -> `Users`
- `Management` -> `Internal Squads`
- `Management` -> `External Squads`
