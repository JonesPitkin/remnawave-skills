# Users

Источник:

- `https://docs.rw/docs/learn-en/users/`

## Create User

- Открыть `Users` -> `Create user`.
- Username может быть произвольным, но без пробелов.
- В `Traffic & Limits` docs явно описывают:
  - `Data limit`
  - `Traffic reset strategy`
- В `Access Settings` docs явно описывают:
  - subscription expiration date
  - привязку к `Internal Squads`
  - при необходимости к `External Squads`

## User Management Tools

В `Edit user` -> `More actions` официально подтверждены:

1. `Detailed Info`
2. `Show usage`
3. `HWIDs and Devices`

Docs также упоминают, что `Detailed Info` содержит в том числе `VLESS UUIDs` и `Happ Crypto links`.
