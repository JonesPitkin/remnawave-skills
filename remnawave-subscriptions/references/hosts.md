# Hosts

Источник:

- `https://docs.rw/docs/learn-en/hosts/`

## Роль Host

- Host — это gateway, который направляет пользователя к Nodes.
- Клиент получает список доступных Hosts и выбирает один из них.

## Create Host

- `Hosts` -> `Create new host`
- `Host visibility` определяет, доступен ли Host пользователям
- `Remark` — отображаемое имя host в dashboard

## Inbound rule

- Для каждого Host можно выбрать только один inbound.
- Inbound выбирается из доступных `Config Profiles`.
